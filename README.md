# Agentic Internship Progress Tracker and Reflection Assistant

## Overview
The Agentic Internship Progress Tracker and Reflection Assistant is an AI-powered system that helps students track internship progress, generate weekly reflections, identify pending tasks, extract skills, and create internship reports automatically.

Built using Langflow, OpenRouter APIs, Python, and SQLite, the system demonstrates core Agentic AI capabilities including tool usage, memory management, and reflection-based reasoning.

## Features
- Weekly internship progress tracking
- AI-generated reflections and learning summaries
- Skill extraction from internship activities
- Pending task detection
- Internship history and memory storage
- Automated report generation
- Mentor performance insights

## Technology Stack
- Langflow
- OpenRouter API
- Python
- SQLite Database
- Agentic AI Workflow

## Workflow
1. Student submits internship updates.
2. Agent analyzes activities.
3. Memory retrieves previous records.
4. Skills and progress are evaluated.
5. Reflection and recommendations are generated.
6. Results are stored and displayed.

## Agentic AI Components
- Current Date Tool
- Message History Tool
- SQL Database Tool
- URL Retrieval Tool
- Reflection and Replanning Mechanism

## Installation
```bash
git clone <repository-url>
cd project-folder
pip install -r requirements.txt
```

Configure your OpenRouter API key and run the Langflow project.
